Download the complete folder called "L'Explorateur Virtual Console Project" and open the .als file. 

The most crucial step when setting up the virtual console is to select the correct MIDI outputs for each of the external instruments within the Ableton devices.

This needs to be done individually for each device. Once the correct MIDI output has been selected, you can save the configuration as a preset in order to easily recall it later.

This step cannot be prepared in advance, as it depends on the MIDI interface being used to send MIDI data to the organ.
